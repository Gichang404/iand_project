> 
![image](https://github.com/seongjiko/iand_project_public/assets/46768743/063bf871-629f-4c3d-9bec-8977fed78bf1)

### 프로젝트 내용
> 아이앤 서비스는 생성형 AI를 활용한 위인 교육 플랫폼이다. 과학자, 수학자, 철학자, 음악가 등의 위대한 업적을 남긴 위인들을 흥미있게 공부할 수 있다.
